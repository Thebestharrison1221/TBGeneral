About TBGeneral

TBGeneral is a powerful plugin designed to enhance your Minecraft server with easy-to-use commands, customizable configuration, and no need for coding.

Installation Guide

Step 1: Download

Visit our GitHub repository to download the latest version of the plugin.

Step 2: Install

Upload the plugin to your Minecraft server's "plugins" directory.

Step 3: Configure

Edit the `config.yml` file to customize the plugin to your liking.

Features

  • Easy-to-use commands

  • No coding required
  • Enhanced server management
